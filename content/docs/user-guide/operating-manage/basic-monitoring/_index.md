# 服务指标

<h2 id="1">介绍</h2>

使用 Prometheus 进行指标采集，然后使用 Grafana 进行数据图形化展示。

Grafana 是基于 JS 开发的，功能齐全的度量仪表盘，可以对接多数据源，实现图形化实时监控。
