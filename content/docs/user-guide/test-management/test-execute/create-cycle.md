# 测试执行

通过本章您将了解到在Choerodon中如何执行测试，查看测试进度。

![](/img/docs/user-guide/test-management/test-execute/test-execute.png)

1. 收缩树状图
2. 筛选循环和阶段信息
3. 循环基本信息
4. 筛选测试执行
5. 执行的阶段名称
6. 直接将执行状态置为通过
7. 点击此按钮可以进入到测试执行页面，可以对测试步骤的结果进行记录，查看执行的详细信息及变更状态。
