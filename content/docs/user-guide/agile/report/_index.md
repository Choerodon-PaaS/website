## 介绍

报告会根据您项目的进展情况以多个维度直观地记录和展示您项目、迭代、版本、进度等汇总情况，其中包含燃尽图、Sprint报告、累计流量图、版本报告、迭代速度图、史诗报告和统计图。

通过本文您将了解什么是报告，以及报告中能您能够得到什么信息？

- **菜单层次**：项目层
- **菜单路径**：敏捷管理>报告
- **默认角色**：项目成员
